Software Checks 1

Check that a backup system is in place and working. It is necessary to make an occasional full backup of the machine PC and regular backups of key data. Key data includes:
  • The welder database files WelderDb.mdf and WelderDb_log.ldf
  • The log database files TFWLogDb.mdf and TFWLogDb_log.ldf
  • Flash removal programs
 REMEMBER: Backup is always the responsibility of your company. TFW cannot make backups for you or restore backups that you haven't made.
Check that WeldArchive is running and appropriately configured to save weld data.
